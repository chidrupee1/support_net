# 🎓 SupportNet – Student Support Platform

SupportNet is a web-based platform designed to connect students with mentors or teachers for academic support, doubt clarification, and guidance.

---

## 🚀 Features
- 👥 Connect students with mentors/teachers  
- 📝 Raise and track academic queries  
- 🔔 Notification system for updates  
- 📊 Structured communication between users  
- 🔐 Role-based access (student/mentor)

---

## 🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript  
- Backend: PHP (XAMPP)  
- Database: MySQL  

---

## 📌 How it works
- Students can post queries or request help  
- Mentors/teachers can view and respond  
- Queries are tracked and managed in a structured way
